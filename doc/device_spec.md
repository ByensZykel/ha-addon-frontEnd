# 设备说明

## 设备类型

* `type == 1` - DIY 设备（`uiid == 1`）

* `type == 2` - LAN 设备

* `type == 4` - CLOUD 设备

* `type == 8` - Zigbee 设备

## 设备 UIID

* `uiid == 1` - 单通道插座

* `uiid == 2` - 双通道插座

* `uiid == 3` - 三通道插座

* `uiid == 4` - 四通道插座

* `uiid == 5` - 单通道功率检测插座
